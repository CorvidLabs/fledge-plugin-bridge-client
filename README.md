# fledge-plugin-bridge-client — fledge plugin

Bridge client - read/write files on connected bridge sessions

## Install

```bash
fledge plugins install ./fledge-plugin-bridge-client
```

Or after publishing:

```bash
fledge plugins install owner/fledge-plugin-bridge-client
```

## Commands

| Command | Description |
|---------|-------------|
| `fledge fledge-plugin-bridge-client` | Bridge client - read/write files on connected bridge sessions |

## Development

Edit `plugin.toml` to configure commands, hooks, and capabilities.
See [fledge plugin docs](https://github.com/CorvidLabs/fledge) for the full plugin format.
